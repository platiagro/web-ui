**Default**
```js

const active = false;

const count = 6;

<LogsButton count={count} active={active}/>;
```

**Selected**
```js 

const active = true;

const count = 6;

<LogsButton  count={count} active={active}/>;
```

**Disabled**
```js 
const disabled = true;

<LogsButton disabled={disabled}/>;
```
